# Action: GenerateUIComponentsView

- Objective: Save the View file "\Views\{Controller}\{View}.cshtml", generating the *Mockup Html Example*.
- Steps:
    1- Read the simplex specification at start of file "\Views\{Controller}\{View}.cshtml"
    2- Create a complete *Mockup Html Example* for the correspondent View inheritance
    3- For each UIComponent create a correspondent complete *Mockup Html Example*
    4- Check that it UIComponent conforms the styles and scripts definded 
    5- Keep the simplex notation at first position in the file content, between HTML comments `<!-- ... -->`.
    6- Check all styles and scripts references were included
- Examples: See examples at `.agent/knowledge/SampleWorkfolder/Views/Seguranca2/ListaUsuarios.cshtml`
