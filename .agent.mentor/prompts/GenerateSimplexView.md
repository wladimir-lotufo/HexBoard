# Action: GenerateSimplexView

- Objective: Save the View file "\Views\{Controller}\{View}.cshtml", generating a simplex View definition.
- Steps:
    1- Verify de correct Controller dataname it belongs
    2- Verify de page standard to be used: StdView1, or StdView2, or Modal2
    3- Update file content setting defined style set 
    4- The simplex notation should be a first position in the file content, between HTML comments `<!-- ... -->`.
- Examples: See examples at `.agent/knowledge/SampleWorkfolder/Views/Seguranca2/ListaUsuarios.cshtml`
