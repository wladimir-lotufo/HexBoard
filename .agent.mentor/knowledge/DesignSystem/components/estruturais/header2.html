ViewHeader

Template HTML

HTML 
                                            <!-- ViewHeader -->
<div class="row">
    <div class="col-lg-4">
        <h4 class="font-extra-bold text-primary">Painel de Usuários</h4>
        <h6>
            <span id="NomeConta">Todos Usuários</span>
            <span id="Divisor"></span>
            <span id="NomeUnidade"></span>
        </h6>
    </div>

    <div class="col-lg-8 pull-right">
        <!-- Optional -->
        <button id="but_NovoUsuario" class="btn btn-default pull-right" onclick="but_NovoUsuario_Click()" type="button" data-toggle="tooltip" data-placement="top" title="Adicionar Novo usuário" data-original-title="Adicionar Novo Usuário">

        </button>
    </div>
</div>

Padrão de Nomenclatura

  • Componente: `ViewHeader` (herda de PanelHeading)
  • Elementos internos: `NomeConta`, `Divisor`, `NomeUnidade` (conforme necessidade)
  • Botões de ação são opcionais na coluna direita

Variações

  • Container pai (`.hpanel`) pode ter borda esquerda customizada: `border-left:solid; border-left-color:#62cb31; border-left-width:5px`
  • Cores de borda podem variar conforme contexto (verde para sucesso, vermelho para alertas, etc.)

Bibliotecas Necessárias

  • Bootstrap (layout grid e classes utilitárias)
  • Font Awesome (ícones)
