Descrição

O ModalContent é o container principal que envolve todo o conteúdo de um StdModal. Ele é o ponto de partida do HTML em um arquivo de modal .cshtml, contendo todos os outros componentes do modal (ColorLine, ModalHeader, ModalBody, ModalFooter).

Quando Usar

  • Como primeiro elemento HTML em um StdModal
  • Logo após o cabeçalho Razor do StdModal
  • Para envolver todo o conteúdo visual do modal

Componentes Filhos

Um ModalContent tipicamente contém:

  • Row - Container Bootstrap grid
  •   └─ Col - Coluna (geralmente col-lg-12)
  •       ├─ ColorLine - Linha decorativa
  •       ├─ ModalHeader - Cabeçalho
  •       ├─ ModalBody - Corpo
  •       └─ ModalFooter - Rodapé

Atributo ID

  • Deve ter um ID único com prefixo mod_
  • Exemplo: id="mod_DetalheUsuario"
  • O ID deve corresponder ao nome funcional do modal

Estrutura Hierárquica Completa

  • ModalPlaceholder (na View principal)
  • └─ ModalContent (div.modal-content com ID)
  •     └─ Row
  •         └─ Col
  •             ├─ ColorLine
  •             ├─ ModalHeader
  •             ├─ ModalBody
  •             └─ ModalFooter

Classes CSS

  • .modal-content - Classe Bootstrap para conteúdo de modal
  • Fornecida por: Bootstrap

Observações Importantes

  • É o primeiro elemento HTML em um arquivo StdModal .cshtml
  • Sempre deve ter um ID único
  • Sempre contém Row > Col como estrutura interna
  • Não confundir com a div.modal-dialog (que fica no ModalPlaceholder)

Bibliotecas Necessárias

  • Bootstrap (modals e grid system)
