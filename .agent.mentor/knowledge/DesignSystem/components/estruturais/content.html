Descrição

O Content é um wrapper de conteúdo usado em StdView para envolver o conteúdo principal da página. Ele adiciona margens laterais automáticas, criando um espaçamento adequado nas bordas da tela.

Quando Usar

  • Logo após o cabeçalho Razor em um StdView
  • Quando você quer margens laterais no conteúdo
  • Para páginas com layout centralizado e espaçado
  • Quando o conteúdo não precisa ocupar toda a largura da tela

Content vs ContentFull

  • Content - Adiciona margens laterais (padding lateral)
  • ContentFull - Ocupa toda a largura disponível (sem padding lateral)
  • Use Content para páginas normais
  • Use ContentFull para grids e tabelas que precisam de mais espaço

Estrutura Hierárquica

  • StdView (cabeçalho Razor)
  • └─ Content (div.content)
  •     └─ Row (div.row)
  •         └─ Col (div.col-lg-*)
  •             └─ UIComponents (Panel, Grid, Form, etc.)

Classes CSS

  • .content - Classe principal do componente
  • Definida em: Content/style.css

Observações Importantes

  • Sempre deve conter pelo menos um Row dentro
  • Não use Content dentro de ContentFull ou vice-versa
  • O Content é aplicado logo após o cabeçalho Razor do StdView

Bibliotecas Necessárias

  • Bootstrap (grid system)
  • style.css (definição da classe .content)
