Descrição

A ColorLine é um elemento decorativo que adiciona uma linha colorida no topo de modais e painéis. É um detalhe visual que ajuda a dar identidade e elegância à interface do sistema.

Quando Usar

  • No topo de modais (logo após a div.modal-content)
  • No topo de painéis especiais
  • Como primeiro elemento dentro de um container
  • Para adicionar um toque visual decorativo

Posicionamento

  • Em modais: logo após <div class="col-lg-12">
  • Antes do ModalHeader
  • Como primeiro elemento visual do componente

Estrutura Hierárquica (em Modal)

  • ModalContent (div.modal-content)
  • └─ Row (div.row)
  •     └─ Col (div.col-lg-12)
  •         ├─ ColorLine (div.color-line)
  •         ├─ ModalHeader
  •         ├─ ModalBody
  •         └─ ModalFooter

Classes CSS

  • .color-line - Classe principal do componente
  • Definida em: Content/style.css
  • Cria uma linha colorida com gradiente

Características Visuais

  • Linha horizontal fina no topo
  • Gradiente de cores (definido pelo tema)
  • Altura mínima (geralmente 3-5px)
  • Largura 100% do container

Observações Importantes

  • É um elemento vazio (self-closing)
  • Não contém texto ou outros elementos
  • Sempre deve ser o primeiro elemento visual
  • Usado principalmente em modais, mas pode ser usado em outros contextos

Bibliotecas Necessárias

  • style.css (definição da classe .color-line)
