Descrição

O ModalHeader é o cabeçalho de um modal, contendo o título e o botão de fechar. Aparece logo após a ColorLine dentro do ModalContent.

Quando Usar

  • Em todo StdModal
  • Logo após a ColorLine
  • Antes do ModalBody

Elementos Obrigatórios

  • <button class="close"> - Botão de fechar (×)
  • <h4 class="modal-title"> - Título do modal

Posicionamento

  • Dentro de ModalContent > Row > Col
  • Após ColorLine
  • Antes de ModalBody

Classes CSS

  • .modal-header - Classe Bootstrap para header de modal
  • .close - Classe Bootstrap para botão fechar
  • .modal-title - Classe Bootstrap para título

Observações Importantes

  • O botão close deve vir antes do título
  • O botão close tem data-dismiss="modal" para fechar o modal
  • Use h4 para o título (padrão Bootstrap)

Bibliotecas Necessárias

  • Bootstrap (modals)
