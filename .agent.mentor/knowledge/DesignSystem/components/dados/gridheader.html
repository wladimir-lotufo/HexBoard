Padrão de Nomenclatura

  • ID (opcional): `grd_header_{NomeColuna}` (ex: `grd_header_Nome`)
  • Prefixo `grd_header_` é recomendado quando usar ID
  • ID é útil para manipulação via JavaScript

Atributos Comuns

  • class="text-center" - Centraliza o texto do cabeçalho
  • class="text-right" - Alinha o texto à direita
  • class="pull-right" - Alinha o elemento à direita
  • width="5%" - Define largura em percentual
  • width="60px" - Define largura em pixels
  • data-toggle="tooltip" - Adiciona tooltip ao cabeçalho

Variações

  • Header padrão: `<th>Nome</th>`
  • Header centralizado: `<th class="text-center">Nome</th>`
  • Header com largura fixa: `<th width="60px" class="text-center">Ações</th>`
  • Header de ações (vazio): `<th width="5%">&nbsp;</th>`
  • Header com tooltip: `<th data-toggle="tooltip" title="Descrição">Nome</th>`

Bibliotecas Necessárias

  • Bootstrap (estilos de tabela)
  • Bootstrap Tooltip (para tooltips, se usado)
