Padrão de Nomenclatura

  • ID: `swi_{NomeCampo}` (ex: `swi_FlAtivo`)
  • Classe: `js-switch` é obrigatória
  • Prefixo `swi_` é obrigatório

Comportamento

  • Para booleanos de dois estados (Sim/Não, Ativo/Inativo)
  • Substitui o checkbox quando se deseja uma metáfora de liga/desliga

Bibliotecas Necessárias
