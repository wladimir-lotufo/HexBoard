Template HTML

HTML 
<a id="but_Voltar" href="#" class="btn btn-default">Voltar</a>

Padrão de Nomenclatura

  • ID: `but_Voltar` (fixo)
  • Prefixo `but_` é obrigatório
  • Elemento: `<a>` tag (anchor) com atributo `href`

Variações

  • Padrão (67%): btn-default btn-lg btn-block
  • Aviso (33%): btn-warning btn-lg btn-block
  • Geralmente usado em formulários de login e configuração
  • Sempre usa texto "Voltar" sem ícone

Bibliotecas Necessárias

  • Bootstrap (estilos de botão)
