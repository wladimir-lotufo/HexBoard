Padrão de Nomenclatura

  • ID: `but_Programar` (fixo)
  • Prefixo `but_` é obrigatório

Variações

  • Padrão: btn-primary btn-sm (mais comum para ações de agendamento)
  • Proeminente: btn-primary (tamanho normal para ações principais)
  • Confirmado: btn-success btn-sm (para itens já agendados)
  • Opcional: btn-default btn-sm (para agendamento secundário)
  • Sempre usa ícone fa-calendar para indicar funcionalidade de agendamento

Bibliotecas Necessárias

  • Bootstrap (estilos de botão)
  • Font Awesome (ícone fa-calendar)
