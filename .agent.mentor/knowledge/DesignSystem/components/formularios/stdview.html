Painel de Campanhas

Exemplo de View usando o template StdView

Este é um exemplo de como uma View criada com o template StdView apareceria no sistema. O conteúdo real da View seria inserido dentro da área de Content ou ContentFull.

Nota: O StdView é um template estrutural que define o cabeçalho Razor da página (.cshtml). O conteúdo visual é adicionado posteriormente usando componentes como Content, ContentFull, Panel, etc.

Template CSHTML

CSHTML 
                                            @{
    ViewBag.Title = "Painel de Campanhas";
    Layout = "~/Views/Shared/_Layout2.cshtml";
}

Descrição

O StdView é o template padrão para criação de Views (páginas/telas) no sistema. Ele herda de View e define a estrutura básica de uma página .cshtml, incluindo o título da página e o layout pai a ser utilizado.

Quando Usar

  • Ao criar uma nova tela (View) no sistema
  • Quando a página precisa do layout padrão do sistema (_Layout2.cshtml)
  • Para páginas que exibem conteúdo dentro da estrutura principal da aplicação

Propriedades do ViewBag

  • ViewBag.Title - Define o título da página (aparece na aba do navegador)
  • Layout - Define o layout pai a ser utilizado (geralmente _Layout2.cshtml)

Componentes de Conteúdo

Após o cabeçalho Razor, você deve adicionar um dos seguintes componentes de conteúdo:

  • Content - Wrapper de conteúdo padrão com margens laterais
  • ContentFull - Wrapper de conteúdo que ocupa toda a largura disponível

Estrutura Hierárquica

  • StdView (cabeçalho Razor)
  • └─ Content ou ContentFull
  •     └─ Row
  •         └─ Col
  •             └─ UIComponents (Panel, Grid, Form, etc.)

Observações Importantes

  • O StdView é apenas o cabeçalho da página - não é um componente visual
  • Sempre deve ser seguido por Content ou ContentFull
  • O layout _Layout2.cshtml já inclui a navegação, header e footer do sistema
  • Para modais, use StdModal ao invés de StdView

Bibliotecas Necessárias

  • ASP.NET MVC Razor (fornecido pelo framework)
  • Layout pai (_Layout2.cshtml) que já inclui Bootstrap, jQuery, etc.
