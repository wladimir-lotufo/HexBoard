Descrição

O StdModal é o template padrão para criação de Modais (janelas popup) no sistema. Ele herda de View e define a estrutura básica de uma View modal .cshtml, incluindo o modelo de dados, título da página e a estrutura HTML do modal.

Quando Usar

  • Ao criar uma janela modal (popup) no sistema
  • Para formulários de edição/criação que abrem em popup
  • Para exibir detalhes de registros sem sair da página principal
  • Quando a View principal precisa ter um ModalPlaceholder

Propriedades do Cabeçalho

  • @model - Define o modelo de dados do modal (ex: ModDetalheUsuario)
  • ViewBag.Title - Define o título da página modal
  • jsonModel - Serialização opcional do modelo para uso em JavaScript

Componentes do Modal

Um StdModal completo contém os seguintes componentes:

  • ModalContent - Container principal do modal
  • ColorLine - Linha colorida decorativa no topo
  • ModalHeader - Cabeçalho com título e botão fechar
  • ModalBody - Corpo do modal onde ficam os UIComponents
  • ModalFooter - Rodapé com botões de ação

Estrutura Hierárquica

  • StdModal (cabeçalho Razor + @model)
  • └─ ModalContent (div.modal-content)
  •     └─ Row
  •         └─ Col
  •             ├─ ColorLine
  •             ├─ ModalHeader
  •             ├─ ModalBody (UIComponents aqui)
  •             └─ ModalFooter

Integração com ModalPlaceholder

A View principal que abre este modal deve ter um ModalPlaceholder correspondente. O ModalPlaceholder é responsável por carregar dinamicamente o conteúdo do StdModal via AJAX.

Observações Importantes

  • O StdModal não inclui as divs externas do Bootstrap modal (modal fade, modal-dialog)
  • Essas divs externas ficam no ModalPlaceholder da View principal
  • O StdModal começa direto com a div.modal-content
  • Use prefixo mod_ para IDs de modais (ex: mod_DetalheUsuario)
  • Para modelos de dados, use prefixo Mod (ex: ModDetalheUsuario.cs)

Bibliotecas Necessárias

  • ASP.NET MVC Razor (fornecido pelo framework)
  • Newtonsoft.Json (para serialização do modelo)
  • Bootstrap (modals, fornecido pelo layout)
  • jQuery (para manipulação do modal)
