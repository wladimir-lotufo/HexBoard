Biblioteca Necessária

  • SweetAlert - Biblioteca para modais de alerta elegantes
  • CSS: ~/bundles/sweetAlert/css
  • JS: ~/bundles/sweetAlert/js

Tipos de Alert

  • success: Mensagem de sucesso (verde)
  • warning: Aviso ou confirmação (laranja)
  • error: Mensagem de erro (vermelho)
  • info: Informação (azul)

Opções Principais

  • title: Título do modal
  • text: Texto da mensagem
  • type: Tipo do alerta (success, warning, error, info)
  • html: Permite HTML no texto (true/false)
  • showCancelButton: Exibe botão Cancelar (true/false)
  • confirmButtonText: Texto do botão de confirmação
  • cancelButtonText: Texto do botão de cancelar
  • confirmButtonColor: Cor do botão de confirmação
  • closeOnConfirm: Fecha ao confirmar (true/false)
  • closeOnCancel: Fecha ao cancelar (true/false)

Exemplo com Callback

swal({
    title: "Confirme?",
    text: "Deseja excluir este item?",
    type: "warning",
    showCancelButton: true,
    confirmButtonText: "Sim",
    cancelButtonText: "Não",
    closeOnConfirm: true
}, function(isConfirm) {
    if (isConfirm) {
        // Ação ao confirmar
        console.log("Confirmado!");
    } else {
        // Ação ao cancelar
        console.log("Cancelado!");
    }
});
