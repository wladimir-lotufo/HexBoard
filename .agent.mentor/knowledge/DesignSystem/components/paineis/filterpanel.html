Padrão de Nomenclatura

  • Classe: `hpanel` (container principal)
  • Classe: `showhide` (botão de expandir/recolher)
  • Classe: `hbuilt` (estilo do cabeçalho)

Variações

  • Pode iniciar recolhido (adicionar lógica JS)
  • Cores disponíveis: hpanel (default), hgreen, hyellow, hblue
  • Panel-footer é opcional (para botões de ação)

Bibliotecas Necessárias

  • Bootstrap (panels)
  • Homer Theme Scripts (funcionalidade showhide)
  • Font Awesome (ícones)
